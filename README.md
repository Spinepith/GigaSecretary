# GigaSecretary

</div>

Интеллектуальный Telegram-бот для автоматизации документооборота в компаниях.
<br>
В основе проекта - искусственный интеллект GigaChat, обученный понимать текст документов, классифицировать их и помогать в их обработке.


### Возможности бота
* Классификация документов - анализирует входящие файлы и определяет их тип.
* Маршрутизация по отделам - предлагает отправить файл в подходящий отдел, основываясь на контексте документа.
* Сравнение с шаблонами - проверяет документы на соответствие установленным требованиям.
* Уведомления и учёт - информирует сотрудников о новых поступлениях и сохраняет данные о переданных документах.


## Как запустить
1.  **Установите зависимости**
<br>Перейдите в терминале в корень проекта и выполните команду.
    ```
    pip install -r requirements.txt
    ```

2. **Создайте базу данных PostgreSQL.**
<br>Используйте файл **db.sql** для создания таблиц.
<br>Заполните таблицы данными. Для этого можете использовать файлы с префиксом **insert_**.

3. Запустите скрипт, который сформирует структуру отделов компании по данным из БД, из корня проекта.
    ```
    python -m scripts.create_department_dirs
    ```

4. [Создайте бота в Telegram](https://www.t.me/BotFather), чтобы получить его токен.
5. Также понадобится токен для использования GigaChat. [Инструкция по получению.](https://developers.sber.ru/docs/ru/gigachat/quickstart/ind-create-project)
6. Откройте файл **config.json**, который находится в корне проекта, и вставьте нужные данные в соответствующие поля.

7. **Запустите бота**
<br>Выполните в терминале из корневой папки проекта.
    ```
    python -m src.main
    ```
 
**Программа написана на Python 3.11**
